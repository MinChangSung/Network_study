SSH(Secure Shell)

소개

암호화된 원격 접속 기능을 제공합니다.

Telnet

주로 라우터나 스위치 같은 네트워크 장비들을 직접 찾아다니며 관리하기 어려워 원격으로 접속할 때 사용하는 기능입니다.

보안상의 취약점으로 현대에는 SSH로 대체된 추세입니다.

PC command

#telnet [ip address]
:텔넷에 접속합니다.

ctrl + ^ + X
:현재 접속한 telnet을 종료하지 않고 잠시 나옵니다.

#sh session
:현재 연결된 telnet 목록을 확인할 수 있으며 목록 번호를 입력하면 해당 telnet에 접속이 가능합니다.

#disconnect [number]
:세션 목록에서 특정 텔넷과의 연결을 종료합니다.

Router command

telnet 설정모드 접속(login 설정을 안하면 password를 묻지 않으니 유의합니다.)
(config)#line vty [minimum] [maximum]
(config-line)#login
(config-line)#password [password]

telnet ACL
(config-line)#access-class [access-list number] [in/out]
