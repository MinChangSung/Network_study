TFTP(Trivial File Transfer Protocol)

소개

라우터의 구성 파일을 미리 만들어서 TFTP 서버에 저장해두었다가 구성 파일을 TFTP로 라우터에게 전송해 라우터를 셋팅하거나 업데이트합니다. 이외에도 IOS 업데이트 파일등을 전송하는 용도로 사용하는 프로토콜입니다.

Command

#copy tftp running-config
:TFTP 서버에 저장된 구성 파일을 라우터의 RAM에 복사합니다.

#copy tftp startup-config
:TFTP 서버에 저장된 구성 파일을 라우터의 NVRAM에 복사합니다.

#copy running-config tftp
:RAM에 저장중인 구성 파일을 TFTP 서버에 복사합니다.

#copy tftp flash
TFTP 서버에 저장된 구성 파일을 라우터의 Flash memory에 복사합니다.
