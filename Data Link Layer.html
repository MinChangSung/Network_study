PDU: Frame

전반적인 역할

  1. MAC(물리) 주소 부여
    :NIC(Network INterface Card)라는 통신 장치에 부여되어 나오며
    각 노드들을 고유하게 식별하는 주소입니다.
  2. Collision(충돌) 도메인 분리
    :2계층 장비는 (Port별로) 충돌 도메인을 분리하여 CSMA/CD로 인한 전송 대기 상태를 축소시킵니다. 충돌의 확장을 차단합니다.
  3. 네트워크 연결 방식 결정
    :중개기 배치, 회선 접속 방식 등을 통해 다양한 네트워크 구조를 구성합니다.

    4. *FCS:이진 신호 전송에서 오류를 감지합니다.

회선 접속 방식

컴퓨터: 연산도 가능 / 단말기: 입출력만 가능

Point to Point Line (Peer to Peer)
:단말기별로 컴퓨터에 1:1 연결

Multipoint Line
:하나의 컴퓨터에 여러 단말기를 버스 구조로 연결

Concentration Line
단말기들의 저속 전송 데이터를 집선 장치가 모아 컴퓨터에 고속으로 전송

Multiplexing Line
:하나의 회선에 다수의 채널 신호를 결합하여 전송

WAN 접속 방식

WAN은 전화국(통신사)등이 구축한 대규모 네트워크를 말하며 이 회선을 이용하는 방식을 요약합니다.

전용선 방식(leased Line)
:이용하고자 하는 WAN 상의 회선을 자신의 전용선으로 임대하는 방식입니다.(PPP)

회선 스위칭 방식(Circuit Switched)
:회선을 통째로 임대하는 것이 아니라 통신을 할 때만 회선을 사용하고 반납하는 방식입니다.(전화)(PPP)

패킷 스위칭 방식(Packet Switched)
:대역폭이 넓은 하나의 회선을 대역폭이 좁은 다수의 채널로 쪼개어 여러 사용자와 공유합니다.(Frame Relay)

세부목록

Bridge(브릿지)

스위치의 조상격 모델이며 소프트웨어 기반, Half-Duplex, 최대 port 수의 제한, 프레임 처리 방식의 제한, 다양한 전송 속도 불가 등의 이유로 거의 쓰이지 않습니다.

*디지털 송수신지

송신지=Source
수신지=Destination

도메인(Domain)

'영역'이라는 뜻입니다.
(브로드캐스트 도메인은 브로드캐스트를 기준으로 나눈 영역을 말합니다.)

Port

통신장비의 interface를 의미합니다. 즉 출입구를 의미합니다.

Link

port와 port를 잇는 경로를 의미합니다.

LAN Segment

4계층 PDU를 segment라고 하는 것과는 다른 개념입니다. 스위치의 한 포트에 연결된 영역을 2계층에서도 segment라고 합니다.

MAC 주소의 범위

LAN 내부에서만 고유 주소로 활용되며 라우터를 통해 다른 LAN과 통신할 때는 source MAC address가 해당 라우터의 MAC주소로 재기록되어 전송됩니다.즉 LAN을 넘어가면 MAC주소는 주소로써 활용되지 못합니다.

NIC의 MAC 변경

MAC 주소는 고유한 주소로 사용하기 위해 고안되었지만 요즘에는 대부분 변경가능합니다.

hybrid topology 예시

partial mesh: 모든 노드가 P2P하는 것이 아니라 필요한 곳끼리만 P2P하는 방식
extended star: 기존의 star의 중개기에 다른 star의 중개기를 P2P연결하여 확장하는 방식
