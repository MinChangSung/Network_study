네트워크란?

정의: 정보를 나눌 수 있는 환경

구성: 노드 + 데이터 + 매체 + 프로토콜

노드(Node)

데이터를 생산하고 소비하는 기초 단위입니다.
(end device, PC, server ...)

데이터(Data)

1.디지털 데이터(이진 신호(on&off))
2.아날로그 데이터(주파수(연속된 파형))

PDU(Protocol Data Unit)
계층마다, 프로토콜마다 정보를 묶는 단위에 차이가 있습니다. 이러한 단위의 형태를 PDU라고 합니다.

매체(Medium)

주요 단위: 대역폭(bandwidth)
대역폭이 클수록 전송 능력(양,속도)이 향상되며
bps(bit per second)를 기본 단위로 합니다.

매체 유형: 회선(유,무선) / 중개기
케이블, 셀룰러... / 라우터, 스위치, 허브, 안테나...

프로토콜(protocol)

(영어)protocol = 규약(국어)

:충돌 방지, 우선 순위 결정 등 통신을 위한 약속 체계

세부목록

  • Cisco CLI
  • Socket Programming

네트워크의 구성을 이론적으로 모듈화

=자기 계층의 데이터(헤더파일)만 해석
