Cable

케이블 종류

copper wire(구리선)
Coaxial Cable(동축 케이블):제작이 어려우나 내구성이 좋습니다.
TP Cable(Twisted Pair Cable): 제작이 쉽고 값이 쌉니다..
-STP Cable(Shield~): 절연체가 있어 상대적으로 무겁습니다.
-UTP Cable(Un~): 절연체가 없어 상대적으로 가볍습니다.
(특정 명칭이 없는 대부분의 케이블은 구리선이라고 봅니다.)

Fiber Optics(광섬유)
:전기가 아닌 빛을 매질로 하며 유리로 제작합니다.
:제작도 유지보수도 어렵지만 구리선과 비교가 안되는 속도에 전자기의 간섭을 받지 않는다는 장점이 있습니다.
-single mode: 한 줄기의 레이저를 매질로 한다. 비싸고 대역폭 높아 장거리 전송이 가능하다.
-multi mode: 여러 갈래의 LED를 매질로 한다. 상대적으로 싸고 근거리 통신용이다.

케이블 규격

대표적인 케이블 규격을 소개합니다.

10 Base T: 10Mbps + Baseband + Twisted Pair Cable (+100m)

10 Base FL: 10Mbps + Baseband + Fiber optic Link (+100m)

100 Base 2: 100Mbps + Baseband + 200m (Thin Cable 사용)

100 Base 5: 100Mbps + Baseband + 500m (Thin Cable 사용)

100 Base TX: 100Mbps + Baseband + UTP cable(Category 5) (+100m)

100 Base T2: 100Mbps + Baseband + UTP cable(Category 3,4,5) (+100m)

100 Base T4: 100Mbps + Baseband + UTP cable(Category 3) (+100m)

100 Base FX: 100Mbps + Baseband + Fiber optic (+2Km~10Km)

1000 Base SX: 1000Mbps + Baseband +Short Wavelen-gth(광케이블) (+270~550m)

1000 Base T: 1000Mbps + Baseband + UTP Cable(Category 5) (+270~550m)

UTP Cable

성능에 따른 분류
Category.1: 전화망 용도로 사용됩니다.(음성만 전달)
Category.2: 4Mbps(안쓰이임)
Category.3: 10 Base T
Category.4: 16Mbps(토큰링 네트워크에서 사용)
Category.5: 100 Base TX(Fast Ethenet) ~ 1000 Base T
Category.6: 1000 Base TX ~ 10G Base(Cat6a)
Category.7: 10G Base
Category.8: 40G Base

기능에 따른 분류
(8가닥의 구리선 배열이 달라집니다.)
Straight-through 케이블: 다른 계층 장비를 연결
Cross-over 케이블: 같은 계층 장비를 연결
Roll-over 케이블: 라우터의 콘솔 접속에 사용
(신형 스위치의 경우 auto-MDIX를 제공하는 포트가 있는데 이걸 쓰면 기능에 따른 분류와 상관없이 모든 utp cable이 호환됩니다.)
